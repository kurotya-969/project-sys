# しすとの30日間 - ビジュアルノベルゲーム

30ターン分岐型ノベルゲーム。プレイヤーは毎日「遊ぶ/仕事/何もしない」の選択を行い、30日以内に目標金額の達成または好感度向上を目指します。

## プロジェクト構造

```
visual-novel-game/
├── index.html              # メインHTMLファイル
├── css/
│   └── style.css          # スタイルシート
├── js/
│   ├── main.js            # エントリーポイント
│   ├── gameState.js       # ゲーム状態管理（実装予定）
│   ├── eventSystem.js     # イベント処理システム（実装予定）
│   ├── uiController.js    # UI制御（実装予定）
│   ├── saveSystem.js      # セーブ・ロード機能（実装予定）
│   └── audioManager.js    # 音響管理（実装予定）
├── assets/
│   ├── images/            # 背景画像・キャラクター立ち絵
│   ├── audio/             # BGM・効果音
│   └── data/              # イベントデータ（JSON）
└── README.md
```

## 技術仕様

- **フロントエンド**: Vanilla JavaScript (ES6+)
- **スタイリング**: CSS3 (Flexbox/Grid)
- **音響**: Howler.js ライブラリ
- **データ保存**: localStorage API
- **モジュール管理**: ES6 Modules

## 起動方法

1. プロジェクトファイルをローカルサーバーで実行してください
2. ブラウザで `index.html` を開いてください

注意: ローカルファイルシステムから直接開く場合、ES6モジュールやCORSの制限により正常に動作しない可能性があります。

## ゲーム仕様

### 基本ルール
- 30日間のターン制ゲーム
- 毎日「遊ぶ」「仕事」「何もしない」から行動を選択
- 好感度と所持金を管理してエンディングを目指す

### エンディング条件
1. **理想の共存エンド**: 好感度70以上 かつ 所持金100,000円以上
2. **夢を叶えるエンド**: 所持金100,000円以上
3. **心でつながるエンド**: 好感度70以上
4. **通常失敗エンド**: 上記条件を満たさない場合
5. **空白エンド**: 「何もしない」を10回連続選択

## ビジュアルシステム

### 背景画像システム
- 3つの時間帯（昼・夕方・夜）の背景を自動切り替え
- 日数に基づいた自動背景変更機能
- フェード効果による滑らかな切り替え
- 画像が存在しない場合のプレースホルダー表示

### キャラクター立ち絵システム
- 複数の表情パターンに対応（デフォルト・嬉しい・悲しい・驚き）
- イベントに応じた表情の自動切り替え
- 表示/非表示の制御機能
- 将来的な立ち絵追加に対応した拡張可能な設計

### 画像アセット配置
画像ファイルは `assets/images/` フォルダに配置してください：
- 背景: `bg_room_day.png`, `bg_room_evening.png`, `bg_room_night.png`
- キャラクター: `sis_default.png`, `sis_happy.png`, `sis_sad.png`, `sis_surprised.png`

詳細は `assets/images/.placeholder_info.txt` を参照してください。
